# Run and deploy your AI Studio app

This contains everything you need to run your app AI locally.

View your app in AI Studio: https://ai.studio/apps/drive/1CYb37TYNJ15gCBeejHDD1aqq2ITAffz1

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
